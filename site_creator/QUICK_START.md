# ⚡ Site Creator 퀵 스타트 가이드

**5분 만에 웹사이트 배포하기**

## 🚀 초고속 시작

### 1️⃣ 프로그램 실행
```bash
cd site_creator
./setup.sh                        # macOS/Linux
# 또는
setup.bat                         # Windows
python ui.py
```

### 2️⃣ 필수 정보 입력
- **도메인**: `yourdomain.com`
- **서버 IP**: `123.456.789.012`
- **서버 PW**: `your_password`
- **Template**: Browse 버튼으로 템플릿 폴더 선택

### 3️⃣ 순서대로 버튼 클릭
1. **폴더 정리** → 2. **HTML 정리** → 3. **업로드** → 4. **서버 설정** → 5. **SSL 설정**
